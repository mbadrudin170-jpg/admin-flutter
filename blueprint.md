# blueprint.md

## Gambaran Umum

Aplikasi ini adalah aplikasi admin untuk mengelola data pelanggan, paket, transaksi, dan lainnya.

## Fitur

*   **Manajemen Pelanggan Aktif**: Menampilkan daftar pelanggan aktif, status pembayaran, dan status paket. Admin dapat menambah, menghapus, dan melihat detail pelanggan aktif.
*   **Manajemen Data Master**: Mengelola data kategori, paket, dan pelanggan.
*   **Manajemen Transaksi**: Mencatat dan melihat riwayat transaksi.
*   **Lainnya**: Menyediakan menu untuk kritik & saran.
